Octav odía, curl.

March 22, 2021

Tras practicar, dejaré por aquí mi chuleta sobre los comandos curl.

Creacion de usuarios

curl -v -k -u admin@pastcorvus.edu:Pimpollo99 -d ‘name=enfin&email=enfin@pastcorvus.edu&passwordPlaintext=Pimpollo99’ https://10.5.2.14/admin/api/v1/boxes

Creacion de dominios

curl -v -k -u admin@pastcorvus.edu:Pimpollo99 -d ‘name=futurecorvus.edu’ https://10.5.2.14/admin/api/v1/domains

Cambiar contraseña

curl -v -k -u admin@pastcorvus.edu:Pimpollo99 -X “PATCH” -d “passwordPlaintext=pimpollito” https://10.5.2.14/admin/api/v1/boxes/enfin@pastcorvus.edu

Borrar cuenta

curl -v -k -u admin@pastcorvus.edu:Pimpollo99 -X “DELETE” https://10.5.2.14/admin/api/v1/boxes/nombre@dominio.edu

Activar/des cuenta

curl -v -k -u admin@pastcorvus.edu:Pimpollo99 –header “Content-Type: application/json” -X “PATCH” –data ‘{“superAdmin”:true}’ https://10.5.2.14/admin/api/v1/boxes/nombre@dominio.edu

Cuota

curl -v -k -u admin@pastcorvus.edu:Pimpollo99 –header “Content-Type: application/json” -X “PATCH” –data ‘{“storageLimit”:20}’ https://10.5.2.14/admin/api/v1/boxes/name@domain.edu // {“countlimit”:X} Para número de mensajes.

Validacion

curl -v -k -u admin@pastcorvus.edu:Pimpollo99 https://10.5.2.14/admin/api/v1/boxes/nombre@dominio.edu/quota

Checks

curl -v -k -u admin@elmeuprimerdomini.org:noposesaquestapassword https://10.28.1.100/admin/api/v1/domains

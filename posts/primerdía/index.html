Primer día, warm up de github.

March 03, 2021

Para empezar, y como he faltado a la última pero primera lección sobre el poste.io, empezaré con el warm up del github pages que el profesor nos ha facilitado e indicado.

El objetivo de esta práctica es el crear un github pages que muestre por pantalla un mensaje que el profesor nos ha indicado.

Los pasos a seguir han sido los siguientes :

1.- Crear un repositorio de github.

2.- Clonar el repositoro.

3.- Crear un index.html con lo que el profesor nos ha pedido.

4.- commit, push.

5.- Activar el github pages de nuestro github.

6.- ¡tachán!

Luego y por último, se ha pasado el link del page al profesor.

March 15, 2021

A día de hoy, la batalla a contar será sobre homebrew y mkcert.

Los requisitos para certificar nuestro servidor, son: git, homebrew y kmcert.

Instalación de Homebrew:

Necesitamos tener GIT de base.

  • /bin/bash -c “$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

  • test -d ~/.linuxbrew && eval $(~/.linuxbrew/bin/brew shellenv)

  • test -d /home/linuxbrew/.linuxbrew && eval $(/home/linuxbrew/.linuxbrew/bin/brew shellenv)

  • test -r ~/.bash_profile && echo “eval $($(brew –prefix)/bin/brew shellenv)” »~/.bash_profile

  • echo “eval $($(brew –prefix)/bin/brew shellenv)” »~/.profile

Instalar MKCERT (para crear los certificados):

  • brew install mkcert

Configuración y obtención de certificado:

  • mkcert -install (La primera vez veremos un unicode muy majo de un zorro aka firefox)

-Editamos el /etc/hosts, añadiendo a nuestra máquina el dominio de nuestro server (como “por si acaso”)

Quintodía

March 09, 2021

Las leyes del tiempo son relativas en este blog: Aunque estos días hayan sido cuando más paliza se le ha dado al github y hugo, ya hay cosas de días anteriores. Pero tengamos en cuenta que hay un proverbio que dice “El tiempo es el mejor autor: siempre encuentra un final perfecto”.

Necesitamos tres cosas:

  • Hugo.

  • Github

  • Homebrew.

Para instalar Hugo, necesitamos Homebrew. Homebrew se instala siguiendo el siguiente link:

https://brew.sh/

Es un poco lioso, pero al final sale.

Para instalar hugo, simplemente habremos de hacer “brew install hugo”.

Comandos básicos de hugo:

-hugo new site nombre_del_site > Nos generará el esqueleto de nuestra “página web”.

Dentro, hay carpetas muy importantes: content, themes, public…

En themes irá nuestro theme que habremos buscado en hugo. (Habremos de hacer un git clone aquÍ).

En content irán nuestras páginas blog:

-hugo new posts/nombre_del_post.md > crearemos un post, que se almacenará en la carpeta content.

-hugo -t nombre_plantilla > toda la info de nuestro blog irá a la carpeta public. Es el contenido estático de nuestra página.

Comandos básicos de github

git add . > añadimos todo el contenido de la carpeta, o algo en concreto.

git commit -m “msg” > prepararemos el contenido, justificando el envío con un mensaje.

git push > empujamos el contenido a nuestro github.

git clone > clonamos un repositorio de github.

Una combinación de hugo y git nos hará tener una buena estructura en github.

Recordemos configurar bien el archivo config.toml de nuestra carpeta de hugo.
